Instructions:
1. Open up two instances of the same flip server
2. Navigate both to folder containing programs
3. Window 2: gcc chatclient.c -o chatclient
4. Window 1: python3 chatserve.py 44450
5. Window 2: chatclient localhost 44450
(or chatclient [correct ip for flip] 44450)
6. Window 1: Specify handle
7. Window 1: Send first message
8. Take turns sending messages, double messages not supported
